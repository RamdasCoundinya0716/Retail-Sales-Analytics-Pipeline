# Retail Sales Analytics Pipeline

## Overview

A batch data engineering project that ingests retail sales data, transforms it into a dimensional model, and generates business insights through SQL analytics and dashboards.

## Objectives

* Build an end-to-end ETL pipeline
* Design a star schema data warehouse
* Generate sales, customer, and product KPIs
* Automate data loading and reporting

## Key Metrics

* Total Revenue
* Total Orders
* Average Order Value
* Top Selling Products
* Revenue by Category
* Customer Retention

## Tech Stack

* Python
* PostgreSQL
* SQL
* Apache Airflow
* Docker
* Streamlit

## Architecture

Raw Data → ETL Pipeline → PostgreSQL Warehouse → Analytics SQL → Dashboard

## Expected Outcome

A production-style analytics platform capable of transforming raw retail data into actionable business insights through automated data pipelines and reporting.
